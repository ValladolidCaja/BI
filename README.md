# BI – Iconos Corporativos

Repositorio público de iconografía e imágenes estáticas utilizadas en los reportes de Business Intelligence de Valladolid Caja Financiera.

## Propósito

Este repositorio está destinado exclusivamente al almacenamiento y distribución de:

- Iconos KPI (flechas, indicadores, estados)
- Elementos gráficos de filtros
- Imágenes institucionales aprobadas por el área de Publicidad

Su finalidad es permitir el consumo directo desde herramientas como Power BI mediante URLs públicas.

## Alcance y Seguridad

- Este repositorio contiene únicamente archivos gráficos (PNG / SVG).
- No incluye datos, código fuente, información confidencial ni lógica de negocio.
- El acceso público es intencional y necesario para su correcta visualización en Power BI Service.

## Uso

Las imágenes deben consumirse mediante su URL directa (https://raw.githubusercontent.com/ValladolidCaja/BI/) y configurarse como `Image URL` dentro del modelo de datos en Power BI.
